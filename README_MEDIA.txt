
Mr Fiesta - Instrucciones de despliegue (GitHub Pages)

1) Estructura del repo:
   /
   ├─ index.html   <-- este archivo
   └─ media/       <-- coloca aquí tus carpetas y archivos de fotos/videos
      ├─ logo-mrfiesta.png
      ├─ 15 años/
      ├─ Animación logo/
      ├─ Chicoteca Básica/
      ├─ Chicoteca en Casa/
      ├─ Chicoteca Ultra Led/
      ├─ Decoración/
      ├─ Experiencia Neón/
      ├─ FOTOGRAFÍA/comprimidos/
      ├─ Hora loca/
      ├─ invitaciones/
      ├─ Juegos/
      ├─ Just dance y Karaoke/
      └─ Pista led/

2) Publicación:
   - Sube index.html y la carpeta media/ al repositorio.
   - En GitHub: Settings → Pages → Source: Deploy from a branch → main (root).
   - Abre la URL de tu GitHub Pages.

3) Notas:
   - Los nombres con espacios/acentos están codificados en la URL (ej: Chicoteca%20Ultra%20Led).
   - Asegúrate de que los videos e imágenes existan con esos nombres exactos.
   - El logo se espera en: media/logo-mrfiesta.png

Éxitos con tu web, ¡a romperla! 🔥
